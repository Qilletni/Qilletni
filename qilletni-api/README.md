# Qilletni API

This repository is the public API for Qilletni. Packages may use this, utilizing [injectable classes](https://qilletni.dev/native_binding/native_bind_factories/) to create Qilletni objects and interact with the system.

For native API use, see the [Native Bindings](https://qilletni.dev/native_binding/introduction/) docs.

Javadocs are available at [https://api.qilletni.dev/](https://api.qilletni.dev/)
